# 💝 给你的小惊喜

一个可爱浪漫的互动网页礼物，送给特别的你。

## ✨ 功能特点

- 🎮 **三个趣味问答**：火锅辣度、奶茶甜度、想念测试
- 🎵 **背景音乐**：星露谷物语《Summer》
- 💌 **许愿表单**：填写心愿，惊喜实现
- 🎨 **浪漫风格**：粉蓝紫渐变 + 可爱动画

## 🌸 页面流程

1. **首页** - "今天想我了没？"
2. **火锅时间** - 选辣度（有小恶作剧哦）
3. **奶茶时间** - 选甜度（按钮会变大）
4. **真心话** - 想念测试
5. **许愿时间** - 填写愿望
6. **完成** - 愿望已收到

## 🚀 使用方法

### 本地打开
```bash
cd girlfriend-gift
python -m http.server 8000
# 浏览器访问 http://localhost:8000
```

### 部署到 Vercel
1. 把整个文件夹拖到 [vercel.com](https://vercel.com)
2. 自动获得 `xxx.vercel.app` 链接

## 📁 文件结构
